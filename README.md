# vue snippets
## Introduction
开发过程中经常会使用到组件，例如：el-data-table、el-form-renderer。每次都要从头写这些组件的配置，非常麻烦，
为了快速的开发（偷懒）😄开发这个插件。

## install
 通过vs code搜索`vscode-vue-snippets`安装
## usage
|简写|内容|
|--|--|
|etc|data table tableConfig|
|efr|form render content|
